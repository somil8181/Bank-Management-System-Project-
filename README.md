**Bank Management System (java)**

This is a  Bank Management System project  developed in java.  It provides basic banking  operations like account Creation , deposit, withdrawl, balance inquiry,
and account management. This project is designed  for learning core java concepts and object = oriented programming.

**Features**
1.  Create new bank account
2.  View  account details
3.  Deposit money
4.  Withdraw money
5.  Check  account balance
6.  Simple  console-based interface
7.  File handling / (or  databse if used)

   **Technologies Used**
1. Java (Core Java / OOP concepts)
2. Scanner class (for input)
3. File handling /  JDBC  (if used)
4. IDE: Eclipse / intelliJ IDEA / vs code

**Learning Outcomes**
1. Java OOP concepts (Encapsulation, Inheritance, Polymorphism)
2. File Handling / Database basics
3. Problem solving skills
4. Real-world project structure understanding

**How to Run**
1. Clone the repository
2. Open in any Java IDE (Eclipse / IntelliJ)
3. Run Main.java (  on  online  compiler or Vs code)
